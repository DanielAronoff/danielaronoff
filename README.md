Welcome to my personal website repository! This site is built with Jekyll and hosted on GitHub Pages.

## About

This repository contains the source code for my personal website at [danielaronoff.github.io](https://danielaronoff.github.io).

## Technologies Used

- Jekyll
- GitHub Pages
- Markdown

## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then navigate to `http://localhost:4000` in your browser.
